git command 정리

basic

  • git init . : 현재 디렉토리에 로컬 깃 레포지토리 생성
  • git add . : 변경된 파일 Staging
  • git commit -m : 커밋 메시지 작성과 동시에 커밋

git config

  • git config --local user.name : username 설정
  • git config --local user.email : user email 설정
  • git config --global http.sslVerify false : ssl 인증 비활성화
